# `sklearn.cross_decomposition`

_Cross decomposition._

### `CCA` <sub>class</sub>

```python
CCA(n_components=2, *, scale=True, max_iter=500, tol=1e-06, copy=True)
```

Canonical Correlation Analysis, also known as "Mode B" PLS.

### `PLSCanonical` <sub>class</sub>

```python
PLSCanonical(n_components=2, *, scale=True, algorithm='nipals', max_iter=500, tol=1e-06, copy=True)
```

Partial Least Squares transformer and regressor.

### `PLSRegression` <sub>class</sub>

```python
PLSRegression(n_components=2, *, scale=True, max_iter=500, tol=1e-06, copy=True)
```

PLS regression.

### `PLSSVD` <sub>class</sub>

```python
PLSSVD(n_components=2, *, scale=True, copy=True)
```

Partial Least Square SVD.

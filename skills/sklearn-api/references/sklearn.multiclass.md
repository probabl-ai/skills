# `sklearn.multiclass`

_Multiclass classification._

### `OneVsOneClassifier` <sub>class</sub>

```python
OneVsOneClassifier(estimator, *, n_jobs=None)
```

One-vs-one multiclass strategy.

### `OneVsRestClassifier` <sub>class</sub>

```python
OneVsRestClassifier(estimator, *, n_jobs=None, verbose=0)
```

One-vs-the-rest (OvR) multiclass strategy.

### `OutputCodeClassifier` <sub>class</sub>

```python
OutputCodeClassifier(estimator, *, code_size=1.5, random_state=None, n_jobs=None)
```

(Error-Correcting) Output-Code multiclass strategy.
